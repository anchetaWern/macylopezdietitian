# Build a Vue SSR Website for Mary Clarice Lopez, RND

Create a professional Vue SSR website for **Mary Clarice Lopez**, a Registered Nutritionist-Dietitian in the Philippines.

Use this reference website for design inspiration only: `https://www.betterbeing.ph/`

Do not copy the exact text, branding, images, layout, or assets. Instead, create an original site with a similar warm, clean, personal, dietitian/health professional feel.

## Tech Requirements

* Use Vue with SSR.
* Use file-based or clearly organized routing.
* Make the site SEO-friendly.
* Use semantic HTML.
* Use responsive design for mobile, tablet, and desktop.
* Use reusable components.
* Use accessible buttons, links, headings, and forms.
* Optimize for fast loading.
* Use placeholder images where professional photos are not yet available.
* Make CTAs link to: `https://seriousmd.com/doc/mary-clarice-lopez`

## Design Direction

The website should feel:

* Warm
* Professional
* Trustworthy
* Gentle
* Modern
* Clean
* Health and wellness focused
* Personal, not corporate

Use a soft nutrition/wellness-inspired palette such as:

* Warm cream / off-white background
* Soft sage green
* Muted beige
* Light peach or blush accents
* Dark green or charcoal text

Use generous white space, rounded cards, soft section backgrounds, and friendly typography.
